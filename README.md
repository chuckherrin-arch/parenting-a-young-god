# parenting-a-young-god
An answer to Dario Amodei's recent article referencing how we survive our technological adolescence. In this article, Chuck Herrin, veteran CISO and risk manager performs a key failure analysis and offers a perspective on two key biological constraints that he posits must be understood, as well as an invitation for collaboration to pressure test the thesis. We can't address problems we don't know about, and we all have blind spots.
